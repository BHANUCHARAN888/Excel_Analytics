**LEFT()** returns a specified number of characters from the beginning (left side) of a text string.
# syntax: =LEFT(text, number_of_characters)

**RIGHT()** is a text function used to extract a specified number of characters from the end (right side) of a text string.
# syntax: =RIGHT(text,number_of_characters)

**MID()** is used to extract a specific number of characters from the middle of a text string starting from a specified position.
# syntax: =MID(text, start_position, number_of_characters)

**LEN()** Returns the total number of characters in a text string, including spaces.
# syntax: =LEN(char)

**TRIM()** Returns after removing whitespaces in both sides.
# syntax: =TRIM(char)

**CONCAT()** Used to combine multiple text strings into a single string.
# syntax: =CONCAT(text," ",text)

**TEXTJOIN()** Used to combine text strings using a separator or delimiter.
# syntax: =TEXTJOIN("",TRUE,'','')
