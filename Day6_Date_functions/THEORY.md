**TODAY()** which returns the today's date.
# syntax: =TODAY(CELL)

**NOW()** which returns the today's date and time.
# syntax: =NOW(CELL)

**YEAR()** which returns year of cell date.
# syntax: =YEAR(CELL)

**MONTH()** which returns the month of cell date
# syntax: =MONTH(CELL)

**DAY()** which returns the date OF cell.
# syntax: =DAY(CELL)

**DATEDIF()** which returns the difference of cell date and today's date.
# syntax: =DATEDIF(CELL)
